# Git Wrap-Up

## Overview
Stage, commit, and push changes safely.

## Steps
1. Review status:
   ```powershell
   git status
   ```
2. Stage intended files:
   ```powershell
   git add <paths>
   ```
3. Commit with a descriptive message:
   ```powershell
   git commit -m "Describe the change"
   ```
4. Push and confirm a clean tree:
   ```powershell
   git push
   git status
   ```
