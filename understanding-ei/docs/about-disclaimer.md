# About / Disclaimer

## About this project

Understanding EI is an independent educational documentation project. It summarizes publicly available Employment Insurance information in plain language.

## Prominent disclaimer

!!! warning "Unofficial and informational only"
    This website is **not** an official Government of Canada, ESDC, Service Canada, or Social Security Tribunal website.

    Content is for **general informational and educational purposes only** and is **not legal advice**.

    Always rely on official sources for current authoritative wording. If you need guidance for your own circumstances, consult a qualified professional.

## Editorial approach

- Neutral tone
- Public-source-first structure
- No insider framing
- No advocacy strategy language

## Branding boundary

This project intentionally avoids government branding mimicry.
