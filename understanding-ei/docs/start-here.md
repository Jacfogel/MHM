# Start Here

## How to use this site

1. Read the [About / Disclaimer](about-disclaimer.md) page first.
2. Review [What EI Is](what-ei-is.md) for baseline context.
3. Use [What EI Cares About and Why](what-ei-cares-about-and-why.md) to understand the big picture.
4. Open a topic page in **Topic Areas** for structured learning.
5. Cross-check with [Official Sources](references/official-sources.md).

## Reading approach

- Start with **overview** sections.
- Use **jump links** on each page to move between sections.
- Keep notes of terms and check the [Glossary](references/glossary.md).

## Safety and scope

This site does not provide legal advice or case strategies. It is an educational guide to public information.
